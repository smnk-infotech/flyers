# Flyers Charitable Trust Website Documentation

## Overview
This repository contains the website code for Flyers Charitable Trust, a charitable organization based in Coimbatore. The website is built using modern web technologies and frameworks to provide an engaging and responsive user experience.

## Technical Stack
- **Framework**: WordPress with Elementor Page Builder
- **Theme**: Astra 4.11.2
- **Primary Languages**: HTML5, CSS3, JavaScript
- **Build Tools**: WP Rocket 3.18.2 for optimization

## Directory Structure
```
flyers/
├── css/               # Stylesheets
├── js/               # JavaScript files
├── images/           # Image assets
├── fonts/            # Font files
└── README/           # Documentation
```

## Key Features
1. **Modern Design**
   - Responsive layout
   - Custom typography using Inter and Plus Jakarta Sans fonts
   - Animated components
   - Mobile-friendly navigation

2. **Performance Optimizations**
   - Minified CSS and JavaScript
   - Optimized image loading
   - DNS prefetching
   - Caching implementation

3. **Integration Points**
   - Google Maps integration
   - Social media connectivity
   - Contact forms
   - reCAPTCHA security

4. **Color Scheme**
- Primary Blue: #060097
- Accent Purple: #c10fff
- Heading Text: #1e293b
- Body Text: #67768e
- Background: #FFFFFF
- Accent Yellow: #ffcd57
